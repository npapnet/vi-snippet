Pickering Interfaces PXI VISA LabVIEW RT Driver
===============================================
30 Jan 2009

To use Pickering PXI switch cards in LabVIEW RT, copy (by ftp) the .inf files
contained in this file's folder to the folder "/ni-rt/system/" on the
LabVIEW RT target and reboot it.

Information about the Pipx40 LabVIEW driver Pipx40.llb can be found in the
"pipx40 LabVIEW ReadMe" file.
